# mac上使用nvs切换node

> 本文作者：[onresize](https://github.com/onresize)

[安装教程参考](https://www.midwayjs.org/docs/how_to_install_nodejs)

#### nvs基本命令

```js
nvs add 18 // 安装18版本node
nvs ls // 查看已按转的node
nvs ls-remote // 查看所有node版本
nvs use 18 // 使用18版本node 
```
